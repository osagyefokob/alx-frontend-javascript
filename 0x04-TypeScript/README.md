# 0x04-TypeScript

Task 1 — Teacher Interface

This folder contains **task_1**, which implements the `Teacher` interface for the ALX TypeScript project.

## Files

- `task_1/js/main.ts` — contains the TypeScript interface and sample objects.  
- `task_1/package.json` — project configuration and dependencies.  
- `task_1/tsconfig.json` — TypeScript compiler settings.  
- `task_1/webpack.config.js` — webpack build setup.

### Build locally
```bash
cd 0x04-TypeScript/task_1
npm install
npm run build
```
